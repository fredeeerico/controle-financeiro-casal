# Sobre o Projeto

## Visão Geral

O **Controle Financeiro do Casal** é um estudo de caso que demonstra como um problema real de organização financeira pode ser transformado em um sistema estruturado de dados.

Embora o resultado final seja um dashboard em Excel, o projeto foi desenvolvido como um fluxo completo de tratamento de dados, envolvendo automação, padronização, modelagem, documentação e validação.

---

## O problema

O controle financeiro manual de contas e cartões de diferentes instituições financeiras gera retrabalho, inconsistências e dificuldade para consolidar informações.

Além disso, importar manualmente faturas e classificar despesas reduz a confiabilidade dos dados e dificulta análises mensais.

---

## A solução

O projeto propõe um fluxo integrado composto por:

1. Organização padronizada dos arquivos.
2. Extração automática de faturas PDF e CSV através de Python.
3. Geração de planilhas estruturadas de movimentações.
4. Consolidação dos dados em uma base sintética.
5. Atualização automática através do Power Query.
6. Análise em um Dashboard interativo construído em Excel.

---

## Objetivos técnicos

Este projeto demonstra conhecimentos em:

- Gestão de Dados
- Análise de Dados
- Engenharia de Dados
- Automação de Processos
- Power Query
- Modelagem de Dados
- Documentação Técnica
- Governança de Dados

---

## Escopo da versão publicada

Para preservar a privacidade das informações financeiras, o repositório utiliza exclusivamente dados sintéticos.

As faturas disponibilizadas simulam documentos reais apenas para permitir testes do extrator e do dashboard.
